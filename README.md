# Hospital Management Database

## Project Overview

This project simulates a hospital management system using SQL Server.

The database manages:

- Patients
- Doctors
- Departments
- Appointments

## Database Schema

### Tables

- Departments
- Doctors
- Patients
- Appointments

## SQL Concepts Used

- Primary Keys
- Foreign Keys
- INNER JOIN
- LEFT JOIN
- Filtering
- Data Retrieval Queries

## Example Business Questions

- Which doctor handled each appointment?
- What appointments belong to a specific department?
- Which patients visited the hospital?
- What fees were generated from appointments?

## Skills Demonstrated

- Database Design
- Data Modeling
- Relational Databases
- SQL Joins
- Query Optimization
